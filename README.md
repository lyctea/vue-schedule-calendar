# vue-schedule-calendar

[demo](https://yscoder.github.io/vue-schedule-calendar/)

## Install

```bash
npm i vue-schedule-calendar -S
```

## Build Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
